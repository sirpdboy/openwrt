### 【关于近期争议事件的公开回应】

大家好。针对近期大家关注的问题，我在此作一次集中、正式的回应。首先，感谢所有人的关注与支持。

一、关于“售卖固件”的澄清：纯属造谣与诬陷
事情经过如下：

2022年疫情期间，我因个人时间较多，投入大量精力编写插件和优化固件。当时，许多朋友自发赞助支持，所有赞助记录均登记在OpenWrt仓库中并公开。后因赞助人数过多，应大家建议，我建立了赞助者交流群，即后来提到的“VIP群”。

建群初期，赞助金额从几元到几十元不等，许多学生朋友也是其中一员。由于人数增长过快，我将进群门槛统一调整为50元，后调整为80元以上。2024年，因遭受恶意攻击，该群暂停开放一年多。2025年，因工作变动后时间充裕，应网友强烈要求，我开始将原有10多个开源插件从LUA版本逐步升级至JS版本，以适配新版OpenWrt 24.10。在此期间，应网友请求，我重新邀请此前赞助100元以上的朋友进群，每次均核对支付截图和有邀请记录，从没有所谓的售卖固件进VIP群。

需要说明的是：
- 1、所有固件均在群内免费提供；
- 2、免费版与群内版本的插件功能上完全一致，仅插件数量有所不同；

当时我正处于JS框架学习阶段，在实现speedtest.js过程中，确实参考了部分代码。但需要说明的是，任何功能的差异都会导致实现方式大不相同，绝非简单复制粘贴或依赖AI即可完成。我的插件功能与其他版本存在明显差异，每一次代码更新、修复都经过反复测试验证，耗费大量心血。我做插件的初衷是出于个人爱好，后在群友期盼下逐渐成为一种责任，激励我不断投入时间，只为让固件中的插件更加简单易用。
